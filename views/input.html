Use custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Validations input

            <input class="basic-input success-input" placeholder="your input goes here...."/>
            <input class="basic-input warning-input" placeholder="your input goes here...."/>
            <input class="basic-input error-input" placeholder="your input goes here...."/>
